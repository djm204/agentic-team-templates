# Definition of Done

Checklist for task completion. Read before marking work complete.

## Checklist

- [ ] Code written and reviewed
- [ ] Tests written and passing
- [ ] `npm run validate:rules` passes (when touching templates)
- [ ] Committed with conventional commit message
- [ ] No secrets or sensitive data in code

## Honesty Checklist

Before saying you're done:

- [ ] Did you actually run the tests?
- [ ] Did they pass?
- [ ] Did you test the happy path AND error cases?
- [ ] Would you ship this to production?
